# Spotify Homepage Clone

A simplified clone of Spotify's homepage using Next.js, Tailwind CSS, and Spotify Web API.

## Features
- Spotify OAuth login
- Display user playlists
- Responsive UI

## Tech Stack
- Next.js
- Tailwind CSS
- Spotify Web API
- NextAuth.js

## Setup
1. Clone the repo
2. Add `.env.local` with your credentials
3. Run `npm install`
4. Run `npm run dev`
